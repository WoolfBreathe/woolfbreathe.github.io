Ubuntu 桌面版 安装 Window 应用

2021-08-20
标签: Ubuntu   Linux  

Wine简介

Wine是一款开源的应用程序，它提供了在类似Unix的操作系统上运行为微软Windows开发的软件的兼容性。

系统： Ubuntu Desktop 20.04 应用： Winehp 6.0

Wine安装

  1. 首先，运行64位体系结构的系统需要启用32位体系结构。
sudo dpkg --add-architecture i386
  1. 然后，下载存储库密钥并将其添加到系统中。
wget -qO - https://dl.winehq.org/wine-builds/winehq.key | sudo apt-key add -
  1. 之后，使用以下命令在系统中启用Wine apt存储库。
sudo apt-add-repository 'deb https://dl.winehq.org/wine-builds/ubuntu/ focal main'
  1. 在Ubuntu 20.04上安装Wine

使用以下命令从apt信息库安装Wine软件包。该–install-recommends选项将在您的Ubuntu 20.04系统上安装winehq稳定版本的所有推荐软件包。

sudo apt update
sudo apt install --install-recommends winehq-stable

在任何情况下，您在安装过程中都会遇到未满足依赖性的错误，请使用以下命令在Ubuntu上使用aptitude安装wine。

sudo apt install aptitude
sudo aptitude install winehq-stable

这将在Ubuntu 20.04系统上安装Wine和所有需要的软件包。

  1. 测试Wine版本

在您的Ubuntu系统上，Wine的安装已经成功完成。使用以下命令检查您系统上安装的Wine版本。

wine --version

  1. 在ubuntu中，右键点击exe安装包，open with wine windows program … 就可以安装了；
  1. 安装完后，桌面有个 ‘微信.desktop’ ，先用文本编辑器打开，如下图所示在‘Exec=env’后添加 ‘GTK_IM_MODILE=ibus QT_IM_MODULE=ibus’ ，保存，解决可能无法输入中文的问题；

中文乱码（框框）

  1. 下载Windows字体
  1. 把Fonts文件夹内所有字体复制到wine的映射目录内
cp Fonts/* ~/.wine/drive_c/windows/Fonts/

输入框内容

方法一

  1. 下载windows的riched20.dll
  1. 将windows的c:/windows/system32/riched20.dll复制到ubuntu下的~/.wine/drive_c/windows/system32/下
  1. 终端执行winecfg，选择函数库标签页，然后在新增函数库顶替里添加riched20，并且编辑为原装优先于内建，重启微信即可。

方法二(未成功)

在终端执行winetricks riched20，让其自动修复即可，但是执行这个指令可能会提示你not found以及连接失败，网络不可达的错误信息，可按以下方法进行，亲测可用。

  1. 下载了两个文件，给出了这两个文件的下载地址：

链接1: W2KSP4_EN.EXE 链接2: InstMsiW

所以这里我们点击这两个连接进行手动下载，然后将下载的W2KSP4_EN.EXE复制到~/.cache/winetricks/win2ksp4 目录下，再次执行winetricks riched20 即可成功，登陆微信输入显示正常。
