Linux安装MySQL

Mysql搭建

1.下载二进制包(Linux-Generic 线上环境一般用二进制包)

解压到目录  /usr/local/mysql

2.增加执行路径

export PATH=/usr/local/mysql/bin:$PATH

3.创建用户

groupadd -r mysql && useradd -g mysql -s /sbin/nologin mysql

4.安装数据库

/usr/local/mysql/script/mysql_install_db --user=mysql --basedir=/usr/local/mysql --datadir=/mysqldata/node1

/usr/local/mysql/bin/mysqld --initialize --user=mysql --basedir=/usr/local/mysql --datadir=/data/mysql

5.修改配置并启动

/usr/local/mysql/bin/mysqld_safe --defaults-file=/usr/local/mysql/cnf/my.cnf &

6.创建用户并分配权限

删除多余用户 删除test数据库

select user,host,password from mysql.user;
delete from msyql.user where user='';
delete from mysql.user where host<>'localhost';
# set password for root@'localhost'=password('123456');
alter user 'test'@'localhost' identified by '123';
flush privileges;

7.停止数据库

/usr/local/mysql/bin/mysqladmin --socket=/data/mysql/mysql.sock shutdown -p

Mysql在线升级

1.下载新软件包

2.关闭mysql实例

mysqladmin -uroot --socket=XXX shutdown

3.修改my.cnf配置参数

4.执行mysql_upgrade升级库

/usr/local/mysql56/bin/mysql_upgrade -uroot --socket=XXXX -s

-s表示只升级系统权限表不升级数据

5.验证是否成功

Mysql连接

1.socket连接

mysql -S/tmp/mysql.sock -uroot -p 
mysql --socket=/tmp/mysql.sock -uroot -p

2.TCP/IP连接

mysql -h127.0.0.1 -P3306 -uroot -p

Mysql帮助

help select
show global variables like 'socket'  查看属性或配置项

MySQL权限

create user 'test'@'localhost' identified by '123';
grant references on testdb.* to developer@'192.168.0.%';  创建外键权限
grant select, insert, delete, update on testdb.* to developer@'192.168.0.%';
grant create, drop, alter on testdb.* to developer@'192.168.0.%' ;
grant ALL PRIVILEGES on testdb.* to developer@'192.168.0.%';

创建root用户

grant ALL PRIVILEGES on *.* to root@'192.168.0.%' WITH GRANT OPTION;

修改密码

1.找不到初始密码可以在my.ini中[mysqld] 添加：

skip-grant-tables

3.先执行：

flush privileges;

4.再执行修改密码命令，可以了：

set password for root@localhost=password('你的密码');
